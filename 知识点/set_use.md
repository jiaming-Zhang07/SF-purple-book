set 中的元素会自动按升序排列
成员函数：
| 成员函数 | 功能描述 | 备注 |
| --- | --- | --- |
| insert(x) | 插入元素 x | 若 x 已存在则不插入 |
| erase(x) | 删除值为 x 的元素 | 返回删除的个数（0 或 1） |
| erase(it) | 删除迭代器 it 指向的元素 | 不可传入 s.end() |
| find(x) | 查找值为 x 的元素 | 返回迭代器，若无则返回 s.end() |
| count(x) | 统计 x 出现的次数 | set 中只能是 0 或 1 |
| clear() | 清空所有元素 | 线性复杂度 |
